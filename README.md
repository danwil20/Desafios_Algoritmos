# Desafíos de Algoritmos 🚀

Este repositorio contiene la solución a los tres primeros desafíos lógicos y de algoritmos. Los programas están desarrollados en **Python 3** y están diseñados para procesar datos de entrada leyendo archivos de texto plano (`.txt`) pasados como argumentos a través de la línea de comandos.

 Requisitos
* Python 3.13.1 instalado.
* Consola de comandos (PowerShell, CMD o Terminal de Linux/macOS).

 Contenido del Repositorio

* **`Desafio1.py`**: (Reconstrucción de oraciones)
* **`Desafio2.py`** (Las catas de Tom)
* **`Desafio3.py`**: (Realidades alternas del cambio)
---

 Cómo Ejecutar los Desafíos

Cada script recibe la ruta del archivo de prueba como el primer argumento (`sys.argv[1]`). 

Para correr cualquiera de los ejercicios, abre tu terminal dentro de la carpeta del proyecto y ejecuta el siguiente comando:


python NombreDelDesafio.py ruta/del/archivo_de_prueba.txt

Ejemplo Práctico (Desafío 2):
Si tienes un archivo llamado pruebas.txt con los casos de los vinos, lo ejecutas así:

python Desafio2.py pruebas.txt
